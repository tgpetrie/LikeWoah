VideoEdit
Forensic video inspection in a single browser tab — bookmark anomalies, detect motion irregularities and hidden pulses, export PDF reports. Runs entirely on-device. No upload, no telemetry, no account.
🖥

Desktop / Workstation

Full forensic toolkit with keyboard shortcuts, sidebar tabs, dense controls, and every analysis module.
  • Adjust sliders, presets, draw tools
  • Lasso, magic wand, selection library
  • Motion analyzer, ELA, Sobel, blur
  • Object tracking, PDF report export
  • Voice & text commands

Mobile / Touch

Streamlined, thumb-zone reviewer for phones and tablets. Long-press to bookmark, voice for everything else.
  • Pinch-zoom, drag-pan, tap to play
  • Long-press video → bookmark
  • Voice command for navigation & clipping
  • Save clip section · motion analysis
  • Install as PWA (Add to Home Screen)

What it can detect

Blown highlights / crushed blacks
False-color exposure regions
Frame diff & motion spikes
Sobel edges, ELA tamper
Laplacian blur / redaction
Motion irregularity & jerk
Periodicity (rhythmic motion)
Pulse / heart rate (rPPG)
Frame-accurate SMPTE timecode
